# GitHub Portfolio Generator (Multi-Agent System)
## Problem Statement
Develop a multi-agent system using Google ADK that generates a professional
GitHub portfolio for a given GitHub username.
## Input
A GitHub username provided by the user.
## Output
A markdown file summarizing the user's top repositories and programming languages.
## Approach
A root agent orchestrates three sub-agents responsible for fetching repositories,
analyzing data, and generating a markdown portfolio.
